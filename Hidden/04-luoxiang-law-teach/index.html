# 罗翔老师法律讲堂

# 我们为什么要做一个诚信的人？

功利主义

我之所以遵循诚信原则是因为诚信能够给我带来好处，至少在一个长远的角度我能赚到更多的钱

道德主义

我之所以遵循诚信原则是因为诚信本身就是正确的，但行好事，莫问前程

调和论

正义本身是好的，而且通常能带来好的结果

道德功利主义

追求一种结果，过程符合功利主义，结果符合道德主义

批注

没有道德调控的功利主义会形成相对主义，相对主义是一切罪恶的源头

# 法治之良法而治

法治有两个层次 第一是良法而治 第二是普遍适用，法律的制定者也需遵从法律的规定。

这个世界中有良善吗？我们相信任何职业一定有其内在的善。 相对主义的伪宽容会导致社会的崩溃，我们一定要抵制相对主义的虚无

批注

警惕历史虚无主义

# 法治之普遍法规

法治的本质是对权力的约束，法律至上。 法家^1的本质就是一种专断的工具，将法律当作一种权力的工具，权力至上。

法治的人性基础是一种现实主义的人性基础，法治不倡导法家的性恶论，人不是工具

所谓的孝、悌、仁、义、忠、信、贞、廉等等，都不过是人道毁弃，而人为标榜出来的东西。人皆循道而生，天下井然，何来大盗，何须圣人。

人是如此的骄傲，尤其是拥有权力的人，总是喜欢别人说你好话，所以人的内心一定有幽暗的成分，人组成的任何机构都有败坏的倾向，不要对人投入过高的期待，也不要对人的事业投入无限的期待，人一定要收到约束。

人性是经不起试探的，我们要用规则来约束人性，让规则释放人性的尊严，压抑人性的幽暗

# 愚人节与虚假消息

这个世界不存在完全的真相和虚假，在法律中，虚假信息是一种根本性失实，法律要超越民众的偏见，当虚假信息造成的社会影响巨大，就需要法律来制裁

# 我们为什么要读书？

如果读书的目的是为了成功，那么很多的书都没必要去读，这是一种功利主义的价值观 我们读书一定要有两种目的，尤其是要有非功利性的读书，坦然接受自己的失败，接受自己的命定，在低迷的时候依然保持勇气，继续前行。 保持渺小，维持谦卑，承认自己的无知是开启智慧的大门，我们不需要完美的人，需要的发自内心的真善美。 读书不是为了炫耀，读书是为了培养智慧，去与人类伟大的灵魂对话

# 聊聊网络喷子与键盘侠

在网络这种匿名的环境中，人们会特别的好辩，会放纵自己好辩的天性，也会放纵自己无知的热情。 知识越贫乏，人们就越自信，吵架是不需要逻辑的。一个标语性思维的人是没有逻辑的，缺乏自己独立思考的能力，所以不要去和他们辨论。 法律会放纵网络喷子吗？侮辱和诽谤是可以定罪的，此类罪行是亲告罪，需要当事人申诉才会进行处理。当事件上升为公众事件，引起恶劣影响，公安机关会处理。 人肉搜索也可能涉及刑法犯罪，侵犯公民个人信息罪，基准刑法三年以下，情节严重七年以上。

# 为什么刑法要严格解释？

刑法和民法的重要区别在于，刑法是必须呀严格解释的。刑法是强大的国家和渺小的犯罪人的对抗，法无明文不规定不为罪不处罚。 刑法是一把双刃剑，一个是惩罚犯罪一个是保障人权。 历史给人们的教训就是人们从来不吸取教训——黑格尔
