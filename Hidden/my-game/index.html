# 序章：英雄的远征

​ 阿达追踪到蓝宝石的下落，来到了萨班帝都城。在王宫门口遭到卫队士兵的阻拦，不允许阿达进入王宫寻找宝石。双方职责所在，互不相让，只好用一场战斗来决定胜负。强大的彩虹城堡护卫军首领阿达，轻松将卫兵们打败，进入王宫寻找宝石。这时隶属魔界黑暗四魔神之一的索蒙突然出现，向阿达发起攻击，阿达措手不及重伤倒地眼前一片漆黑……

​ 彩虹城堡阿达家里，阿达的弟弟阿凯从恶梦中惊醒过来。他预感到哥哥正处在危险中。起床来到楼下，和母亲谈话得知国王将继续派人去寻找彩虹宝石。阿凯于是去王宫见国王，要替哥哥完成寻找宝石的使命和打听哥哥的下落。国王答应了阿凯的要求，并给他200银币做路费。临走前，阿凯找好朋友丽莎帮忙照顾母亲，不料丽莎定要跟着阿凯一同去寻找宝石，阿凯只好答应了，两人穿过传送之门，来到亚辛大陆东北地区，开始了漫长的冒险旅程。

​ 出村向东北走，来到秋月森林中，在森林尽头击败森林守护者得到了百灵树油。回到村中交给切亚，制成了引路火把。出村向东南方走，来到山脚下。在亚古山洞口使用引路火把，穿越漫长的亚古山洞。 穿过山洞，到达一片辽阔的地方。

# 序章：星之冥河

通关项链获取方法

  1. 出传送庙~右上角靠近山脉有一朵花~上去踩一脚~打开物品栏~会发现多了一个中瓶治疗剂~
  2. 回传送庙~拿左边的试练之剑~右边的水晶甲不要拿~再切记切忌~
  3. 出传送庙~走下面绕一圈来到刚刚这朵花的下面靠近树边有一朵花，这朵花左边踩一下~2个通关项链到手！

​ 出了传送殿，可以看到两个回复神庙，有需要的可以在这附近先练级，但本人觉得没必要，见怪打怪就可以了。 ​ 沿路走，到左上角就是冥河之村，在村内的最左上角有一个人，你过去，等他一走开你可以进到里面的花丛，他大概三十秒走开一次。走到最里面可以拣到一个圆月盾，然后出来，在最左上角的房间里有个馒头师傅，记住他，以后有用。然后去村长家，就是带六角星的屋子，跟他说完话他会给你三百银，然后可以去道具店买个残月草，以备不时之需。 ​ 出村，向右，走到头向下左有一个试炼之洞，建议到三级阿凯会了治疗术再进去会比较安全，在洞里完全逛一遍可以拿到金刚甲，龙牙盾，月光剑和残月草，不要错过。然后出来，大概就四级了(记得见怪打怪，最好不要逃，这样才会不知不觉升级)，出来后向上，就会遇到两个魔界守卫，有治疗术打他们就小意思，打死它们两个就可以进入魔界了。

# 第一章：魔导士的故乡
