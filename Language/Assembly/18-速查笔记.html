# 数字编码

数字编码 正数 负数
[X]原=符号位~数值位 正数的符号位是 0 负数的符号位是 1
[X]反 正数[X]反=[X]原 负数[X]反=符号位保持~数值位取反
[X]补 正数[X]补=[X]原 负数[X]补=[X]反+1
运算 负数[X]原=[[X]补]补
原码：用一个二进制位来表示符号，剩下部分表示数值
[X1]机=01010100
[X1]真值=+1010100=+84
反码：为了把减法运算转换为加法运算，引入了反码和补码
正数[X]反=[X]原
负数[X]反=符号位保持~数值位取反
补码：补码和”模“有关，任何有“模”的计量器，均可化减法为加法运算
[A]补+[B]补=[A+B]补
[A+B]补为正数时，[A+B]补的数值即为真值
[A+B]补为负数时，对[A+B]补的数值进行“求反+1”得到真值

# 寄存器结构

通用寄存器 16bit
AX 累加器 存放中断控制字 乘(除)法运算作为乘(除)数
BX 基址寄存器 查表时做基址（表相对数据段的偏移地址）
CX 计数寄存器 处理字符串和移位指令时，作为计数器
DX 数据寄存器 常用于输入输出
段寄存器 16bit
CS 程序段寄存器 指向程序段在内存中的基地址
DS 数据段寄存器 指向数据段基地址，操作字符串时放源串基址
ES 附加数据段寄存器 指向附加段基地址，操作字符串时放目的基址
SS 堆栈段寄存器 指向堆栈段基地址
IP 指令指针寄存器 程序运行时自动变化，指向下一条指令的地址
SI 补充 bx 做偏移地址寄存器 字符串运算时做源串变址
DI 补充 bx 做偏移地址寄存器 字符串运算时做目的变址
BP 基址指针寄存器 指明数据在堆栈段中的偏移地址
SP 堆栈指针寄存器 指向栈顶位置
标志寄存器 16bit
FR 标志寄存器 16 位寄存器，9 个有效位存放状态标志和控制标志

# 指令系统

# 数据传送 (DST 目的操作数 SRC 源操作数)

数据传送指令
MOV DST,SRC move 赋值指令 8bit 或 16bit 寄存器<=8bit 或 16bit 寄存器或立即数
PUSH SRC push 入栈指令 只能是对“字”操作，高位放高字节，低位放低字节
POP DST pop 出栈指令 只能是对“字”操作，8086 满递减堆栈，栈顶 FFH
XCHG DST,SRC change 交换指令 8bit 或 16bit 寄存器<=>8bit 或 16bit 寄存器
IN IO,SRC input 输入指令 端口数据传寄存器，寄存器<--端口地址
OUT DST,IO output 输出指令 寄存器数据传端口，寄存器-->端口地址
XLAT TABLE 查表指令 AL<=DS:[BX+AL]，需指定 AL 为下标，BX 为表的基地址
假设数据段DS基地址是：2000H，BX获取表格相对DS的基地址是：0100H，那么实际上BX指向的单元在整个内存中的地址是：20100H；
MOV BX,OFFSET TABLE		BX<--表格对DS的偏移地址(又称表格的基地址)
IN AL,1					AL<--从地址为1的端口读取的数据
XLAT TABLE				AL<--DS:[BX+AL]
OUT 5,AL				地址为5的端口输出的数据<--AL

# 地址传送 (REG 寄存器，MEM 内存)

地址传送指令
LEA REG,MEM Load Effective Address 传送偏移地址 REG<=MEM 的偏移地址
LDS REG,MEM Load pointer into DS 传送地址指针 段地址传 DS，偏移传 REG
LES REG,MEM Load pointer into ES 传送地址指针 段地址传 ES，偏移传 REG

# 标志传送

标志传送指令（PSW 标志寄存器）
LAHF Load AH with flag AH<=PSW 低 8bit
SAHF store AH with flag PSW 低 8bit<=AH
PUSHF Push flag PSW 入栈
POPF Pop flag 出栈传 PSW

# 算术运算

加法
ADD DST,SRC 加法指令 DST <= DST+SRC
ADC DST,SRC 带进位加法 DST <= DST + SRC + CF
INC DST 自加一指令 DST <= DST + 1
减法
SUB DST,SRC 减法指令 DST <= DST - SRC
SBB DST,SRC 带借位减法 DST <= DST - SRC - CF
DEC DST 自减一指令 DST <= DST - 1
NEG REG 求补指令 REG<= 0 - REG ，负数[X]补=[X]反+1
CMP DST,SRC 比较指令 DST - SRC，只影响标志寄存器，不改变数值
乘法
MUL SRC 无符号乘法 (AX) <=(AL8) * (SRC8), (DX:AX) <= (AX16) *(SRC16)
IMUL SRC 带符号乘法
除法
DIV SRC 无符号除法 (AL%AH) <= (AX16)/(SRC8), (AX%DX)<=(DX:AX)/(SRC16)
IDIV SRC 带符号除法
拓展
CBW byte-->word 将 AL 保持原值（有符号数）扩展到 AX，8-->16，按最高位补相同数
CWD word-->double 将 AX 保持原值（有符号数）扩展到 DX,AX，16-->32，按最高位补相同数

# 逻辑运算

基本逻辑
AND DST,SRC 按位“与” DST <= DST AND SRC
OR DST,SRC 按位“或” DST <= DST OR SRC
NOT DST 按位“非” DST <= NOT DST
XOR DST,SRC 按位“异或” DST <= DST XOR SRC
TEST DST,SRC 测试指令 DST AND SRC，只影响标志寄存器
移位运算 将 SRC 顺序移动 1 位或 CL 寄存器中指定的位数
SHL SRC 1/CL 逻辑左移
SAL SRC,1/CL 算术左移
SHR SRC,1/CL 逻辑右移
SAR SRC,1/CL 算术右移，算数右移时，最高位保持不变
ROL SRC,1/CL 循环左移
ROR SRC,1/CL 循环右移
RCL SRC,1/CL 带进位循环左移，原先状态位 CF 的值移入 SRC 的最低位
RCR SRC,1/CL 带进位循环右移，原先状态位 CF 的值移入 SRC 的最高位
串操作
REP MOVSB/MOVSW 串传送 [(ES:DI)]<=[(DS:SI)]
REP CMPSB/SMPSW 串比较 [(ES:DI)]-[(DS:SI)]
REP SCASB/SCASW 串搜索（匹配） AL-[(ES:DI)]
REP STOSB/STOSW 串存储 [(ES:DI)]<=AL
REP LODSB/LODSW 串读取 AL<=[(DS:SI)]
字符串传送，比较，搜索(匹配)，本质上是循环，[(ES:DI+ADD)]<=[(DS:SI+ADD)] CX从100到0 ADD根据DF标志决定自增还是自减
MOV SI,1000H			设置源串偏移地址
MOV DI,2000H			设置目的偏移地址
MOV CX,100				设置字符串长度
CLD 					方向标志复位，DF=0，地址向高处增长
REP MOVSB				开始传送字符串

# 程序控制

跳转指令
JMP ADDRESS 无条件转移 无条件跳转到 ADDRESS
JZ/JC/JO/JS/JP EX 当 ZF /CF/OF/SF/PF= 1 时，跳转到 EX
JNZ/JNC/JNO/JNS /JNP EX 当 ZF /CF/OF/SF/PF= 0 时，跳转到 EX
JCXZ EX 测试转移 若 CX = 0，则跳转到 EX (计数器转移，循环)
循环指令
LOOP 指定 CX 数值后循环 CX 数值次数
LOOPE/LOOPZ 需满足 ZF=1 时循环
LOOPNE/LOOPNZ 需满足 ZF=0 时循环
中断
INT n CPU 转入 n 号中断服务子程序
IRET 中断返回指令 退出中断程序返回主程序断点处
子程序
CALL EX/NEAR IP 入栈，子程序的入口地址的位移量 -->IP，CPU 执行子程序
RET （Num） 将栈顶元素出栈 --> IP，将 SP 向高地址移动 Num 位
标志操作
CLC/STC/SMC CF 清零/置 1/取反
CLD/STD DF 清零/置 1
CLI/STI IF=0 不允许中断/IF=1 允许中断
处理器操作
HLT 使处理器处于停止
WAIT 使处理器处于等待
NOP 空操作，用于延时
LOCK 封锁总线指令，可前缀
ESC 让协处理器取指令

# 伪指令

符号定义(类似 C 语言宏定义)
SIGN EQU num/SRC sign 被赋值为 num/SIGN被定义为SRC寄存器
SIGN = num sign 被赋值为 num，可重复使用
数据定义
DATANAME DB num1,num2.... 分配字节空间，同时初始化数据
DATANAME DW num#16 分配字空间，同时初始化数据
DATANAME DB ? 分配空间，不进行初始化
DATANAME DB 'string' 分配空间，按地址递增放入 ASCII 码
DATANAME DB NUM DUP (num) 分配 NUM 字节空间，重复初始化 num
过程定义
PRONAME PROC FAT/NEAR 过程名不能省略，过程类型为 FAR 或 NEAR
RET 返回被调用处
PRONAME ENDP 表示过程结束
CALL PRONAME 调用过程（子程序）
ORG ADDRESS 指定 IP 到 address，后面代码的起始偏移地址为 address
